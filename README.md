# Little RPG

A simple command line RPG made to better understand classes, interfaces and inheritance in Java. The game is made of 10 levels of increasing difficulty, with turn based combat and the ability for the player to attack and heal (for a maximum of 3 times per playturn).

## What I learnt
- Concrete classes
- Abstract classes and when to use them
- When to implement interfaces and why an interface contract is preferable to using single inheritance

## Installation steps

1. Clone the repo in a folder
```shell
git clone https://github.com/giuseppe-lisi/little-rpg
```

2. cd into the src directory `cd little-rpg/src` and run the main method with `java App.java`

3. Play through CLI
